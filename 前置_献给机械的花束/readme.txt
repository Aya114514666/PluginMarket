操作台 (Console) 是机器人导入 NBT 方块时所使用的“控制台区域”，它目前被定义为一个 11*5*11 的全空气区域。

您有责任确保这个区域内都是空气，并且没有生物和其他方块的干扰。
除此外，您有责任确保机器人的背包不会受到其他命令方块的干扰，例如在线奖励或 replaceitem 等。
如果可能，我们建议你在导入期间关闭命令方块，因为任何干扰都可能造成严重问题。

另外，NBT 导入是交给另外一个名字叫做 43c4f567 的机器人进行的，你需要确保它具有管理员权限。
